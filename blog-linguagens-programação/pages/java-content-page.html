    História do Java

    Uma linguagem de programação versátil que é conhecida por sua portabilidade, pois os programas escritos em Java podem ser executados em diferentes plataformas sem a necessidade de recompilação. É comumente usada para desenvolvimento de aplicativos Android, sistemas empresariais e em grandes empresas.
    A história do Java começa em 1991, na empresa Sun Microsystems, onde foi formado um grupo de programadores para criar algo revolucionário. Eles achavam que num futuro próximo, as residências e seus aparelhos teriam ligação com os computadores, era a ideia de casa e dispositivos inteligentes que temos hoje em dia. Nesse time estava o James Gosling, que é reconhecido como o criador do Java. O time foi chamado "Green Team", um trocadilho com "dream team" (time dos sonhos) e green people (alienígenas)

    Primeiros passos com Java

    Java é uma das linguagens de programação mais conhecidas do mundo. Desenvolvida em 1991, pela empresa Sun Microsystems, podemos encontra-la rodando em dispositivos como tablets, smartphones, computadores, etc.
    Além de uma linguagem de programação, o programador Java também pode contar com um conjunto de ferramentas que oferecem soluções para situações comuns de desenvolvimento.
    Basicamente, quem está começando aprender Java, precisa seguir 6 passos básicos:

    Iniciar pela lógica de Programação

    Desenvolver a lógica de programação é fundamental, independente da linguagem que se deseja aprender. Desenvolvendo uma boa lógica de programação possibilita encontrar o melhor jeito de escrever um código, para o computador interpretar corretamente. Possibilitará que você se comunique com a máquina a partir de uma linguagem seja lá qual for bastando se adaptar com a sintaxe específica. Comece desenvolvendo a sua lógica resolvendo exercícios sobre lógica. Até mesmo alguns jogos ajudam, não apenas jogos eletrônicos, mas também jogos de carta, jogos de tabuleiro, etc.

    Estude os conceitos

    Utilize a internet ou até mesmo compre algum livro sobre Java e dê bastante atenção a parte teórica. É fundamental que você conheça os principais conceitos da linguagem para começar. Na internet você encontrará muitos materiais excelente. Neste link você pode conferir o material oficial com a documentação para ir se familiarizando com o assunto.

    Faça download das ferramentas

    Outro ponto extremamente importante que é pré-requisito para começar a programar em Java, é instalar em seu computador o Java JDK (Java Development Kit) e o JRE (Java Runtime Environment).
    Você também deverá instalar uma IDE para escrever seu código Java. Existem várias opções: Eclipse, NetBeans. Escolha o que preferir e faça a instalação.

    Utilize o Java Tutorial

    Existe um portal onde você encontrará o que você precisar para começar a programar em Java. Neste portal você encontrará diversas lições sobre assuntos ligados à programação em Java. Acesse clicando neste link.

    Realize pesquisas em boas fontes

    Para aprender Java você precisará buscar por boas fontes de pesquisa. Leia livros sobre a linguagem recomendados por profissionais mais experientes, pesquise em sites especializados, faça parte de fóruns específicos.
    Algumas dicas de fóruns brasileiros: GUJ e Java Free.

    Faça cursos online

    Aprender Java será muito prazeroso para você. Quando conseguir construir as primeiras aplicações você verá isso. Mesmo que seja algo simples, mas quando você visualizar as possibilidades ficará muito animado para aprender novos assuntos.
    Para auxiliar nesta sua fase de estudos, também é interessante, além de ler bons materiais, entender como profissionais mais experiente trabalham. Para isso faça cursos online de bons instrutores. Atualmente fazer um curso online traz muitas vantagens para o aluno, como por exemplo poder aprender sem sair de casa, além de contar com o apoio de um profissional qualificado para lecionar as aulas.
